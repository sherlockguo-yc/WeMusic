# 更新日志

## 2026-07-03 大规模优化迭代

### 🛡️ 安全修复

| 问题 | 修复 |
|------|------|
| `webhook.js` 硬编码密钥 | 改为从 `process.env.WEBHOOK_SECRET` 读取，未配置则拒绝请求 |
| `/api/play/stream` 无认证 | 新增 `?token=` 查询参数可选认证，前端自动附带 JWT |
| 播放日志无范围校验 | `duration` 限制 ≤86400s（24h），`played_sec ≤ duration` |
| 登录密码太弱 | 最小长度 4→8 位 |
| 管理员权限隔离 | 普通用户访问 admin 端点返回 403 |

### ⚡ 性能优化

| 项目 | 改动 |
|------|------|
| `stats.js` 周报/月报 ~200 行重复 SQL | 抽取 `buildReport()` 公共函数，各路由从 ~110 行缩减到 ~25 行 |
| `lyrics.js` 轮询定时器常驻 | 从 `initLyrics()` 移除，改为面板 `show` 时启动、`hide` 时清除 |
| `qqmusic.js` 废弃函数 | 删除 `getAlbumSongs()` |

### 🐛 Bug 修复

| 问题 | 修复 |
|------|------|
| `main.js` `$ is not defined` 报错 | 补充 `import { $ } from './utils.js'` |
| ESC 关闭浮层顺序错误 | 候选弹窗优先于歌词详情页关闭 |
| `donate-qr.png` 404 | 修正为 `donate-qr-cropped.png` |
| `history.pushState` 猴子补丁 | 删除补丁，改为 `navPush()` 内直接调 `updateNavArrows()` |
| 歌词源无法取消屏蔽 | 新增「已屏蔽的歌词源」折叠区 + 恢复按钮 |

### 🆕 新功能

#### 手机扫码访问
- **设置面板**底部「移动端访问」→ 点击获取 QR 码（含 Wi-Fi 提示）
- **直接访问** `http://localhost:5174/qr`
- **npm run mobile**：手机 USB 连接 + adb → 自动在手机浏览器打开

#### 管理员面板重设计
- **侧边栏入口**：分隔线 + 盾牌图标，仅管理员可见
- **用户列表**：用户名、头像首字母、最近登录时间、注册时间、歌单数、歌曲数
- **使用统计**：7 天 / 30 天 / 总计播放时长
- **反馈管理**：查看/删除用户反馈
- `scrips/build-restart.js`：一键构建 + 重启（`npm run deploy`）

### 📱 移动端 UX

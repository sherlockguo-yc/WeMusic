# 淡入淡出（Crossfade）

## 概述

切歌时通过双音频引擎 + 双 GainNode 实现平滑过渡，当前曲目淡出同时下一曲目淡入，消除切歌的"断崖感"。

## 行为规则

| 场景 | 行为 |
|------|------|
| 自动切歌（播完下一首） | 触发淡入淡出 |
| 手动切歌（上一首/下一首按钮） | 立即取消淡入淡出，直接切换 |
| 单曲循环 | 不触发（playMode === 'single'） |
| 展开视频（mode === 'iframe'） | 不触发 |
| 定时停止 | 不触发（先于淡入淡出判断） |
| 开关关闭 | 退化为常规切歌 |

## 实现

### 架构

```
bgAudio (当前曲) ──→ MediaElementSource ──→ GainNode (当前曲) ──┐
                                                                  ├──→ AudioContext.destination
_nextAudio (下一曲) ──→ MediaElementSource ──→ GainNode (下一曲) ──┘
```

淡入淡出期间：
- 当前曲 GainNode 从当前音量 `linearRampToValueAtTime` 到 0
- 下一曲 GainNode 从 0 `linearRampToValueAtTime` 到目标音量（含归一化增益）

过渡结束后：
- bgAudio 设置新歌 URL，从 0 开始播放（不 seek 到 _nextAudio 的位置）
- _nextAudio 暂停静音
- 加载新歌歌词（方案 B：过渡期间保持旧歌词）

### 进度条（`_tick`）在 crossfade 期间的特殊行为

- 用 `_crossfadeStartTime`（crossfade 开始时的时间戳）合成 elapsed，不从 `bgAudio.currentTime` 读取（后者是旧歌位置）
- `totalDur` 不会被 `bgAudio.duration` 覆盖（保留 `startTimer` 设置的新歌时长）
- 停滞检测（stall）和暂停恢复逻辑在 crossfade 期间跳过
- 接管后 `_crossfading = false`，_tick 恢复正常：从 `bgAudio.currentTime` 读 elapsed

### 关键防竞态机制

- `bgAudio.ended` 事件：crossfade 期间直接忽略（旧歌结束由 crossfade 定时器接管，防止双重 autoAdvance）
- `_applyNormVol` 的 `if (!_crossfading)` 守卫：防止在 crossfade 进行中覆盖 `linearRampToValueAtTime` 的 ramp 值
- `_completeCrossfade` 中 `_crossfading = false` 必须在 `_applyNormVol()` 之前执行，否则增益不会被恢复

### 关键技术点

- `linearRampToValueAtTime`：音频线程插值，非 JS 定时器离散跳变
- 归一化增益：下一首的 gain 在淡入淡出启动时异步拉取（`_fetchGainForNext`）
- 歌词方案 B：`_completeCrossfade` 中调用 `loadLyrics`，过渡期间旧歌词照常同步
- 安全阀：用户手动切歌时 `_cancelCrossfade()` 立即恢复增益并停止 _nextAudio
- 兜底：`_completeCrossfade` 内部 1.5 秒超时强制接管（canplay 未触发时）

### 设置

| localStorage key | 默认值 | 说明 |
|-----------------|--------|------|
| `wemusic_crossfade_enabled` | `'0'`（关闭） | 开关，`'1'` 启用 |
| `wemusic_crossfade_duration` | `'5'` | 过渡时长（秒），可选 3/5/8/10 |

### 修改文件

- `public/index.html`：`<audio id="audioNext">` + 设置面板 toggle + 时长按钮
- `src/player.js`：`_startCrossfade` / `_completeCrossfade` / `_cancelCrossfade`、`autoAdvance` 集成、双 GainNode 管线
- `src/settings.js`：toggle 交互 + 时长选择

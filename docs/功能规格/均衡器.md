# 均衡器（预设 EQ）

> 状态：✅ 已实现 · 实施日期：2026-07-20

## 概述

5 段预设均衡器，基于 Web Audio `BiquadFilterNode`，在两根 GainNode 合并后统一处理。全局生效，选后对所有歌曲生效，直到切换或关闭。

## 预设曲线

| 预设 | 60Hz | 250Hz | 1k | 4k | 12k | 听感 |
|------|------|-------|-----|-----|-----|------|
| 关闭(flat) | 0 | 0 | 0 | 0 | 0 | 原声直出 |
| 流行 | +2 | +1 | +1 | +2 | +2 | 均衡偏亮 |
| 摇滚 | +4 | +2 | -2 | +2 | +4 | V 字形，高低两头翘 |
| 古典 | +1 | +1 | 0 | 0 | -1 | 接近原声，略温暖 |
| 人声 | -3 | -2 | +4 | +2 | 0 | 削弱伴奏，突出人声 |
| 电子 | +5 | -1 | -3 | +1 | +5 | 重低音 + 高音 |

## 频段配置

| 频段 | 类型 | 频率 | Q 值 |
|------|------|------|------|
| 60Hz | lowshelf | 60 | — |
| 250Hz | peaking | 250 | 1.0 |
| 1kHz | peaking | 1000 | 1.0 |
| 4kHz | peaking | 4000 | 1.0 |
| 12kHz | highshelf | 12000 | — |

## 音频管线

```
bgAudio → _gainNode ──┐
                        ├──→ [60Hz] → [250Hz] → [1k] → [4k] → [12k] → destination
_nextAudio → _nextGain ──┘
```

两根 GainNode 合并后统一经过 EQ 链，确保淡入淡出期间的混合输出也走 EQ。

## 设置

| localStorage key | 默认值 | 说明 |
|-----------------|--------|------|
| `wemusic_eq` | `'flat'` | 预设名称，可选 flat/流行/摇滚/古典/人声/电子 |

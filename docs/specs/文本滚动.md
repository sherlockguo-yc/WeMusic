# 文本左右滚动

## 设计目标

底栏歌名、底栏视频标题、浮窗视频标题在文字过长时自动左右滚动。

## 实现方案

**rAF 状态机驱动**（非 CSS keyframes），统一速度 40px/s：

```
暂停 2s → 滚动 → 暂停 2s → 回滚 → 重复
```

所有滚动位置用「父级 `overflow: hidden` 裁剪 + 内层 span `translateX`」。

}

检测溢出：`inner.scrollWidth > parent.clientWidth + 4`。

## 三处位置

| 位置 | 滚动原理 | 触发 |
|------|----------|------|
| 底栏歌名 | grid 列 60-180px，inner span 在列内 translateX | `checkMarquee($('npTitle'))` |
| 底栏视频标题 | absolute 定位在进度条和按钮之间，`left`/`right` 设两端 | `checkMarquee(statusInner)` |
| 浮窗视频标题 | flex:1，inner span translateX | `setVpTitle(title)` |

共用一个 `startMarquee(inner, extra)`，三处速度一致。

## 常见问题

- **不滚动**：inner span 不能设 `min-width: max-content`；确保双帧 rAF 后再检测溢出
- **文字被 `…` 截断**：父级必须移除 `text-overflow: ellipsis`（`.marquee` class 处理）
- **标题空白**：`startVideo` 的 `displayTitle` 有歌名兜底，`setVpTitle` 默认 "Bilibili 播放"

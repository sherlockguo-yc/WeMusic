# 数据迁移与导入

## 概述

音乐用户的歌单沉淀在多个平台，支持从外部平台导入歌单降低迁移壁垒。

## 已实现

- **QQ 音乐歌单导入**：解析分享链接，提取歌曲列表。（`POST /api/music/parse-playlist` → `server/services/qqmusic.js`）
- **网易云音乐歌单导入**：解析分享链接，提取歌曲列表。（同上端点 → `server/services/netease.js`）
- **JSON 文件导入/导出**：WeMusic 自身格式的备份与恢复。

## 后续方向

- 网易云"我喜欢的音乐"导入（需要 cookie/token，仅浏览器端处理）
- 导入后标记数据来源（song 表增加 `source_platform` 字段）
- 迁移报告：显示成功/失败匹配数
- 通用 TXT/CSV 格式导入（歌名-歌手）

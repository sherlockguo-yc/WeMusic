# 音乐收藏夹增强

## 概述

当前的"我喜欢"是扁平列表，歌曲多了之后很难管理。目前已实现按歌手聚合视图，后续可通过标签系统和智能歌单进一步提升整理效率。

## 已完成

- **按歌手聚合**：收藏列表支持按歌手分组展示，每个歌手名下折叠显示其所有收藏歌曲。点击歌手可展开/收起歌曲列表，方便按歌手快速定位和浏览。

## 待实现

- **标签系统**：手动打标签（如 #摇滚 #治愈 #开车听），多标签支持，侧边栏按标签筛选，标签颜色自定义。
- **自动标签**：基于 QQ 音乐流派/风格字段自动打标签，基于播放时段打时间标签。
- **智能歌单**：基于标签+规则的动态歌单（如"标签=#摇滚 AND 时长<5:00"），实时计算无需手动维护。

## 思考

按歌手聚合是投入产出比最高的第一步：无需新增数据库字段，直接利用现有歌曲的歌手信息做分组展示，就能显著提升浏览效率。标签系统在此基础上进一步精细化：实现简单（几个数据库字段 + UI），但能显著提升大规模收藏的可用性。智能歌单让用户从"维护歌单"变成"定义规则"。

# 管理功能

> 面向管理员的完整后台能力。当前实现极其单薄（仅用户数/反馈数/总时长 + 用户列表 + 反馈删除，权限仅靠环境变量白名单），需系统性补全。

---

## 一、现状诊断

| 维度 | 现状 | 差距 |
|------|------|------|
| 权限模型 | `ADMIN_USERNAME` 环境变量白名单 | 无角色分级、无操作粒度控制 |
| 数据看板 | 三个静态数字 | 无趋势图、无用户行为分析、无可视化 |
| 用户管理 | 列表展示，只读 | 无法禁用/启用、无详情页、无批量操作 |
| 内容审核 | 仅反馈列表可删除 | 屏蔽源不可管理、无审核工作流 |
| 系统配置 | 手动改 `.env` 重启 | 无运行时配置面板、无功能开关 |
| 操作审计 | 无 | 无操作日志、安全事件黑盒 |
| 系统监控 | 无 | 服务器/API/数据库状态完全不可见 |

---

## 二、角色与权限体系

### 角色定义

| 角色 | 核心能力 | 限制 |
|------|---------|------|
| **超级管理员** | 全部模块 + 管理管理员 + 备份恢复 | 环境变量兜底，不可被 Web 端降级/删除 |
| **管理员** | 除「权限管理」「备份恢复」「功能开关」外的全部模块 | 不可删除/降级超级管理员 |
| **内容审核员** | 仅内容审核 + 用户标记/备注 | 不可禁用/删除用户 |
| **只读观察员** | 数据看板 + 用户列表 + 审计日志（只读） | 不可执行任何修改操作 |

| 查看用户列表/详情 | ✓ | ✓ | ✓(列表) | ✓ |

## 三、数据看板

- **实时概览卡片**：DAU/MAU、新增注册、活跃播放用户数、播放歌曲/歌单/反馈/屏蔽源总量
- **趋势图表**：用户增长折线图、播放时长时段分布热力图、注册→活跃转化漏斗、留存率、反馈趋势
- **内容分析**：热门歌曲 Top 50、热门歌手 Top 20、高频搜索词、播放源健康度、红心/收藏排行
- **用户画像**：活跃度分层、偏好分布、人均使用时长/播放数
- **数据导出**：CSV/JSON 导出、定时周报/月报自动推送、自定义时间范围

---

## 四、用户管理增强

- **详情页**：完整播放历史、歌单明细、反馈历史、登录历史（时间/IP/UA）、操作时间线
- **操作**：禁用/启用账号、重置密码、强制下线、编辑用户信息
- **删除**：级联删除或匿名化关联数据、可选先导出再删除、须超级管理员二次确认
- **批量操作**：条件筛选、批量导出、批量禁用/启用、批量发送站内通知
- **标记与备注**：内部备注、状态标记（正常/观察中/已警告/已封禁）、按标记筛选
- **列表增强**：服务端分页、可排序列、用户名模糊搜索

---

## 五、内容审核

- **屏蔽源管理**：屏蔽源列表、批量取消屏蔽、屏蔽原因分类、自动标记规则、屏蔽源统计
- **歌单审核**：公开歌单审核队列、举报处理、官方精选标记/置顶/推荐位
- **评论审核**：审核队列、自定义敏感词库、用户禁言/解禁、举报处理
- **头像/昵称审核**：违规头像标记→强制重置、敏感昵称检测→强制修改
- **审核工作流**：待审核队列排序、审核记录、驳回必填理由、批量审核

---

## 六、系统配置

- **注册控制**：开放注册/仅邀请制/关闭注册、邀请码管理、IP 黑白名单
- **功能开关（Feature Flags）**：运行时启用/禁用模块、灰度发布（按百分比或用户组）
- **全局参数**：播放默认值、API 限频、缓存策略、推荐参数调整
- **公告管理**：创建/编辑/删除公告、定时发布+过期自动下架、目标用户范围筛选
- **备份与恢复**：手动/自动备份、从备份恢复（须超级管理员验证，自动创建快照）

---

## 七、操作审计

- **管理员操作日志**：操作人/目标/类型/时间/IP、变更前后对比、不可篡改
- **安全事件日志**：登录/登出记录、异地 IP 检测、异常时段操作、API 异常统计
- **日志查询与导出**：多条件筛选，CSV/JSON 导出
- **保留策略**：可配置保留天数，超期自动清理

---

## 八、系统监控

- **服务健康**：CPU/内存/磁盘使用率、Node.js 进程指标、数据库连接池、慢查询
- **API 性能**：P50/P95/P99 响应时间、QPS 趋势、错误率分布、第三方 API 成功率
- **后台任务**：任务面板、失败告警与重试、单次任务耗时
- **告警规则**：可配置阈值、通知方式（站内/邮件/Webhook/企业微信机器人）

---

## 九、核心交互流程

**内容审核闭环**：用户举报 → 系统聚合 → 审核员受理 → 确认屏蔽/驳回 → 通知举报人 → 写入审计日志

**运营日常巡检**：看板概览 → 发现异常 → 趋势详情 → 定位根因 → 导出周报

**功能灰度发布**：超级管理员启用功能开关 → 按比例灰度（20%→50%→100%）→ 观察指标 → 全量/回滚

**用户数据合规**：管理员搜索用户 → 导出数据 → 用户确认 → 二次验证删除

**备份与灾难恢复**：每日自动备份 → 恢复前校验 → 创建快照 → 二次验证 → 执行恢复

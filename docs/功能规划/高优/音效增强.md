# 音效增强

## 概述

当前播放器是纯原声输出，没有任何音频后处理。加入基础音效能力可以显著提升听感，特别是对于音质差异较大的 B 站视频源。

## 功能方向

- **淡入淡出**：✅ **已实现**（2026-07-20）。切歌时平滑过渡（crossfade），独立 toggle 开关（默认关闭），可配置过渡时长（3/5/8/10 秒）。基于双 `<audio>` + 双 GainNode + `linearRampToValueAtTime` 实现采样级连续过渡。仅自动切歌（播完下一首）生效，手动切歌取消过渡。过渡期间歌词保持旧歌（方案 B），过渡结束切换到新歌歌词。不适用场景：展开视频、单曲循环。
- **均衡器（EQ）**：预设曲线（流行/摇滚/古典/人声/电子）+ 自定义 10 段图形 EQ。
- **音量标准化**：✅ **已实现**（见 `功能规格/音量归一化.md`）。基于 EBU R128 进行响度归一化，解决不同 B 站视频源音量差异问题。2026-07-13 架构改造后全场景生效（不再仅后台）。

## 思考

均衡器和淡入淡出都可以用 Web Audio API（`BiquadFilterNode` + `GainNode`）在前端实现，无需后端改动。

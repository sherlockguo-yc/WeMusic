# 音效增强

## 概述

播放器已具备音量归一化、淡入淡出、预设 EQ 三项音效能力。

## 功能方向

- **淡入淡出**：✅ **已实现**（见 `功能规格/淡入淡出.md`）。切歌时平滑过渡（crossfade），独立 toggle 开关（默认关闭），可配置过渡时长。
- **均衡器（EQ）**：✅ **已实现**（2026-07-20）。5 段预设 EQ（流行/摇滚/古典/人声/电子），基于 Web Audio BiquadFilterNode，全局生效。
- **音量标准化**：✅ **已实现**（见 `功能规格/音量归一化.md`）。基于 EBU R128 进行响度归一化，解决不同 B 站视频源音量差异问题。2026-07-13 架构改造后全场景生效（不再仅后台）。

## 思考

均衡器和淡入淡出都可以用 Web Audio API（`BiquadFilterNode` + `GainNode`）在前端实现，无需后端改动。

# 音乐笔记 / 听歌日记

## 概述

允许用户为歌曲写私人笔记，记录听歌时的心情、想法、场景。把听歌变成记录生活的方式。

## 功能方向

- **歌曲笔记**：Markdown 笔记，关联时间戳 + 心情标记（开心/悲伤/平静/兴奋/怀旧），按时间/心情/歌手筛选。
- **听歌日记**：每日自动生成听歌摘要，用户可追加内容，导出为 Markdown。
- **记忆回顾**："去年的今天你听了什么"、"这首歌你第一次听是 2023 年 3 月 15 日"，随机推送旧笔记。

## 思考

技术实现简单（文本+元数据存储），但体验价值很高。和现有播放统计互补：统计是量化数据，笔记是质化记忆。建议在播放页底部放小图标入口，hover 展开。

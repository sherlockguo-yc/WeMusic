# 睡眠 / 专注模式

## 概述

在定时停止基础上加入环境音、白噪音、番茄钟，让 WeMusic 帮助睡眠和专注工作。

## 已实现

- **定时关闭 + 倒计时显示**（✅ 已上线）：设置 15/30/60 分钟或播完当前歌曲后停止播放。设置后底部显示实时倒计时（`14 分 37 秒后停止`），每秒刷新。定时到期调用 `stopPlayback()` 销毁播放器。实现位置：`src/settings.js` 的 `setSleep()` / `clearSleep()` / `updateSleepHint()`。

## 功能方向

- **环境音/白噪音**：内置雨声/海浪/篝火/森林/风扇/咖啡店，可叠加音乐同时播放（独立音量控制），无缝循环。
- **番茄钟专注模式**：25 分钟专注/5 分钟休息，自动切换专注歌单/放松音乐，完成统计。
- **睡眠增强**：定时音量渐弱（最后 5 分钟逐渐降低），睡眠专属 UI（暗色/极简/无蓝光），睡眠统计。

## 思考

让 WeMusic 从"听歌工具"拓展为"生活方式工具"。环境音用两个 Audio 元素叠加实现。白噪音文件可从 freesound.org 获取。

# 自动化与快捷指令

## 概述

让用户通过 URL Scheme、PWA 快捷方式、自定义快捷键快速触发播放器操作，减少操作步骤。

## 功能方向

- **URL Scheme**：`/play/song/123` 直接播放、`/search/晴天` 直接搜索、`/radio` 开启私人 FM。
- **PWA 快捷方式**：右键 PWA 图标 → "播放日推"/"打开我喜欢"。
- **键盘快捷键自定义**：支持多键组合，预设配置方案（Vim 风格/标准风格）。

## 思考

实现成本低、收益大。特别是 URL Scheme 和快捷键自定义，能显著提升重度用户效率。

# KTV / 卡拉 OK 模式

## 概述

结合已有视频播放和歌词功能，增加全屏视频 + 逐字高亮歌词 + 人声抑制。

## 功能方向

- **全屏 KTV 模式**：视频全屏背景、大字歌词、逐行/逐字高亮、横竖屏支持。
- **人声抑制**：简易立体声相位消除 → AI 人声分离（Spleeter/Demucs，需 GPU）。
- **趣味评分**：基于音高检测的简易评分系统。

## 思考

实现复杂度高。人声抑制是核心挑战：简单相位相减法效果差，AI 分离效果好但需 GPU 推理。逐字高亮需歌词带逐字时间戳（绝大多数网易云歌词不提供）。**建议分阶段：全屏歌词 + 基础人声消减 → AI 人声分离 → 评分系统。**

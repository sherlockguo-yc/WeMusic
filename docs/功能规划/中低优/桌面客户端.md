# 桌面客户端（Electron / Tauri）

## 概述

将 WeMusic 从纯 Web 应用打包为桌面客户端，解锁浏览器沙箱做不到的事情。

## 解锁能力

- **全局快捷键**：不依赖页面焦点，系统级快捷键。
- **桌面歌词**：真正的置顶悬浮窗。
- **系统托盘**：迷你播放器、快速控制。
- **开机自启动**：配合定时开机播放当闹钟。
- **本地文件系统**：直接读写本地音乐文件。

## 思考

这是**交付形态**的变化，不是功能本身，但会推动多个功能（桌面歌词、全局快捷键、定时开机播放）从"部分可用"变为"完整可用"。Electron 开发成本低但包体积大，Tauri 更轻量但 Rust 有学习成本。

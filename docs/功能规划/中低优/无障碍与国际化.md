# 无障碍与国际化

## 概述

扩展可达用户群体：屏幕阅读器用户、非中文母语用户、移动端用户。

## 功能方向

- **屏幕阅读器 A11y**：`aria-label`、键盘导航、`aria-live` 播报、高对比度主题。
- **国际化 i18n**：提取 UI 文本到语言文件，初期中文+英文，后续日文/韩文。
- **移动端手势**：响应式布局、滑动切歌、长按快进、下拉刷新。

## 思考

基础设施类功能，不直接产生新能力但能扩大用户群。**建议在日常开发中渐进式补充，每次改 UI 顺手加 aria-label 和 i18n key。**

## 待确认

- [ ] i18n 方案：纯前端 JSON vs 服务端返回翻译
- [ ] 移动端：单独 UI 布局 vs CSS 响应式
- [ ] 高对比度主题：独立选项 vs 跟随系统

# 音频可视化

## 概述

在播放页面加入实时音频频谱/波形可视化，增强沉浸感和视觉吸引力。

## 功能方向

- **频谱分析仪（Spectrum）**：经典柱状频谱，多种配色（跟随主题色/彩虹/单色）和样式（柱状/波浪线/圆形雷达）。
- **波形图**：整首歌的波形预览（类似 SoundCloud），可点击跳转，已播放部分变色。
- **粒子/氛围效果**：频谱驱动的动态粒子动画，可替代静态封面作为播放页背景。

## 思考

纯前端功能，用 Web Audio API 的 `AnalyserNode` 即可实现。频谱分析仪实现最简单（几十行代码），效果直观。波形图需用 OfflineAudioContext 预分析。**建议路径：频谱 → 波形 → 粒子效果。**

# 本地音乐 / 离线缓存

## 概述

目前所有音频都依赖 Bilibili 在线流，每次播放都需要请求音频代理。如果 B 站视频被下架或网络不稳定，歌曲就无法播放。支持离线缓存能大幅提升可用性。

## 功能方向

- **离线缓存**：已播放过的歌曲自动缓存音频流到本地（IndexedDB 或 OPFS）。缓存管理：查看大小、按歌曲/时间清理、一键清空。弱网/离线自动降级到缓存播放。
- **本地音乐上传**（二期）：支持上传 MP3/FLAC/WAV/AAC，自动解析 ID3 标签，与 QQ 音乐元数据匹配。

## 思考

离线缓存更轻量：歌曲元数据已在服务端，只需缓存音频流。本地音乐上传需要处理文件上传、ID3 解析、存储空间管理，复杂度高不少。**建议：先做离线缓存，再做本地上传。**

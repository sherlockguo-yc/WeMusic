# 外部服务集成

## 概述

和主流音乐社区/工具打通，让 WeMusic 的播放数据同步到其他平台，扩大生态影响力。

## 功能方向

- **Last.fm Scrobbling**：播放记录自动上报 + Now Playing 状态同步。
- **Discord Rich Presence**：状态栏显示当前歌曲（歌名+歌手+封面+进度条）。
- **ListenBrainz / Libre.fm**：开源替代方案。
- **Webhook 通知**：播放事件时发 POST 请求，联动 Home Assistant/IFTTT（如播放音乐时自动调暗灯光）。

## 思考

低成本高回报：实现简单（HTTP API 调用），但给用户"数据不锁死在 WeMusic"的信心。Last.fm 和 Discord 用户量大、API 文档完善，优先做。

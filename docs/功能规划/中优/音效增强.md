# 音效增强

## 概述

当前播放器是纯原声输出，没有任何音频后处理。加入基础音效能力可以显著提升听感，特别是对于音质差异较大的 B 站视频源。

## 功能方向

- **淡入淡出**：切歌时平滑过渡（crossfade），可配置过渡时长（0/3/5/8/10 秒）。
- **均衡器（EQ）**：预设曲线（流行/摇滚/古典/人声/电子）+ 自定义 10 段图形 EQ。
- **音量标准化**：基于 EBU R128 或 ReplayGain 进行响度归一化，解决不同 B 站视频源音量差异巨大的问题。

## 思考

均衡器和淡入淡出都可以用 Web Audio API（`BiquadFilterNode` + `GainNode`）在前端实现，无需后端改动。**淡入淡出对体验提升最直接、实现成本最低，建议作为第一步。**

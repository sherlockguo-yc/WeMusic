# 歌单模板与曲库健康

## 概述

帮助用户维护干净的音乐库：检测重复歌曲、修正元数据错误、提供歌单模板。

## 功能方向

- **重复歌曲检测**：检测同歌不同视频源、跨歌单重复，合并建议保留最佳视频源。
- **歌单模板**：预设"华语经典 100"/"欧美热单"/"日系 ACG"等，一键创建。
- **死链检测**：定期检查 B 站视频可用性，标记不可用，一键换源。
- **元数据修正**：允许用户手动修正不准确的元数据（歌手名/专辑信息）。

## 思考

面向收藏了大量歌曲的重度用户。当歌单 500+ 首歌时，手动找重复和死链几乎不可能。元数据修正是争议功能——有维护成本但 QQ 音乐确实存在数据质量问题。

## 待确认

- [ ] 死链检测频率：每天/每周
- [ ] 歌单模板数据来源：硬编码 vs 实时拉取排行榜

# 电台 / 漫游模式

## 概述

"我不知道听什么"是音乐播放器最常见的痛点。电台模式可以在用户不主动选歌时自动播放合适的歌曲。

## 功能方向

- **歌曲电台**：以当前歌曲为种子，基于歌手/流派/时长等维度找相似歌曲，支持点赞/点踩调整方向。
- **歌手电台**：播放该歌手及其相似歌手的歌曲，优先热门→穿插冷门→相似歌手。
- **流派/风格电台**：预设摇滚/电子/爵士/古典/说唱等频道。
- **私人 FM**：综合长期偏好的无限电台，已喜欢的歌降权，基于时段调整风格。

## 思考

核心难点是"相似歌曲"计算。初期可做简单版本：同歌手热门歌 + 同流派热门歌混排，播放量做多样性采样。用户量大了之后协同过滤更有效。

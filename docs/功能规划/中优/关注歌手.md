# 关注歌手

## 概述

用户关注喜欢的歌手，当歌手发布新歌时收到通知。从"被动搜索"到"主动发现"。

## 功能方向

- **关注歌手**：歌手页面一键关注，已关注列表（侧边栏入口），歌手动态（新歌/新专辑）。
- **新歌通知**：定时轮询关注歌手的最新歌曲，浏览器通知推送，支持每日/每周汇总。
- **歌手上新日历**：日历视图展示当月/当周新歌，点击直接播放。

## 思考

价值取决于 QQ 音乐 API 是否能稳定返回"最新歌曲"数据。如果只能拿到全部歌曲列表，需本地差分计算。PWA + Service Worker 可实现离线通知。

## 待确认

- [ ] 轮询频率：每小时/每天/每周

# 智能推荐增强

## 概述

现有的推荐系统已经用了播放历史 + 喜欢 + 歌单 + 完播率的协同思路，可以进一步挖掘场景化推荐和每日推荐。

## 功能方向

- **每日推荐歌单（日推）**：每天自动生成 20-30 首歌，考虑最近7天未播放、偏好流派、播放时段偏好。日推历史可回溯。
- **场景/心情歌单**：预设场景标签（运动/学习/睡前/通勤），自动生成对应推荐。
- **相似歌曲**：播放页底部展示"相似歌曲"列表，一键加入队列。

## 思考

日推是网易云音乐的核心功能，用户黏性极高。但需要足够的数据积累才有意义。实现上可复用现有推荐引擎的大部分逻辑，主要是时间窗口和新鲜度权重的调整。

## 待确认

- [ ] 日推的生成时机：每天凌晨定时生成 vs 用户首次打开时生成
- [ ] 相似歌曲的数据来源：纯本地计算 vs 外部 API
- [ ] 场景标签：AI 模型还是简单规则匹配

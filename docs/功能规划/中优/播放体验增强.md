# 播放体验增强

## 概述

在现有播放控制基础上增加专业用户（学歌、扒谱、练琴）和日常用户都需要的实用功能。

## 功能方向

- **AB 循环**：在进度条上标记 A/B 点，循环播放区间。快捷键 `[` 设 A、`]` 设 B。典型场景：学歌、乐器练习跟弹。
- **变速播放**：0.5x ~ 2x，保持音调不变（pitch-preserving）。典型场景：慢速听细节、快速浏览长歌。
- **定时开机播放**：设置指定时间自动开始播放（当闹钟），音量渐强（fade in 30s/60s）。

## 思考

AB 循环和变速播放都可以用 Web Audio API 实现。Chrome 的 `<audio>` 标签已原生支持 pitch-preserving 变速。定时开机播放作为 Web 应用有一定局限，配合 PWA + Periodic Background Sync 可能实现。

## 待确认

- [ ] 变速播放：浏览器原生支持足够还是需要 Web Audio 额外处理
- [ ] 定时开机播放：是否需要桌面客户端才能实现
- [ ] AB 循环 UI 放哪：进度条上标记 vs 播放控制区

# 音乐收藏夹增强

## 概述

当前的"我喜欢"是扁平列表，歌曲多了之后很难管理。标签系统和智能歌单可以提升整理效率。

## 功能方向

- **标签系统**：手动打标签（如 #摇滚 #治愈 #开车听），多标签支持，侧边栏按标签筛选，标签颜色自定义。
- **自动标签**：基于 QQ 音乐流派/风格字段自动打标签，基于播放时段打时间标签。
- **智能歌单**：基于标签+规则的动态歌单（如"标签=#摇滚 AND 时长<5:00"），实时计算无需手动维护。

## 思考

标签系统投入产出比极高：实现简单（几个数据库字段 + UI），但能显著提升大规模收藏的可用性。智能歌单让用户从"维护歌单"变成"定义规则"。建议先做手动标签，再做自动标签和智能歌单。

# 歌词增强

## 概述

当前已有 LRC 滚动歌词和换源功能，在展示质量和便捷性上可进一步提升。

## 功能方向

- **双语歌词/翻译**：如果网易云歌词源提供了翻译，展示双语对照。允许用户手动添加/编辑翻译。
- **桌面歌词**：迷你悬浮歌词窗口，始终置顶（Web 版用迷你窗口方案，完整支持需桌面客户端）。
- **纯音乐检测**：自动判断纯音乐跳过歌词搜索，判断依据：歌名关键词 + 歌词源返回空 + 时长特征。

## 思考

双语歌词和纯音乐检测实现成本低且不依赖打包，优先做。桌面歌词在 Web 应用中的实现受限——浏览器无法创建置顶悬浮窗，真正实现需要 Electron/Tauri。

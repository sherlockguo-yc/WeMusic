# 协作歌单

## 概述

多个用户共同编辑同一个歌单，实时同步所有修改。类似 Spotify Collaborative Playlist。

## 功能方向

- **创建协作歌单**：开启协作模式，生成邀请链接/码，设置协作者权限（可编辑/仅查看/可添加不可删除）。
- **实时同步**：WebSocket/SSE 同步添加/删除/排序，冲突处理（LWW + 操作日志），"正在编辑"状态指示。
- **协作动态**：操作历史（谁在什么时候添加/删除了什么歌），支持 @某人 添加歌曲建议。

## 思考

协作歌单是社交功能的自然延伸。可和社交功能一起规划：MVP 先做协作歌单 + 歌单公开分享，再做评论/关注。

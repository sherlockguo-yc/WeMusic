# 内容审核

- **屏蔽源管理**：屏蔽源列表（视频源 / 歌词源），按歌曲、屏蔽者、时间筛选。批量取消屏蔽。屏蔽原因分类（视频下架 / 音质差 / 内容不匹配 / 用户举报 / 其他）。自动标记规则（如同一视频被 N 个用户主动换源，自动标记「待审核」）。屏蔽源统计（原因分布饼图、增长趋势）。
- **歌单审核**（社交功能上线后适用）：公开歌单审核队列（标题/封面/简介违规检测）。举报处理工作流（接收 → 核实 → 处理 → 通知举报人）。官方精选歌单标记/置顶/推荐位管理。
- **评论/互动审核**（社交功能上线后适用）：评论审核队列。自定义敏感词库管理。用户禁言/解禁（按时间段或永久）。举报处理闭环。
- **用户头像/昵称审核**：违规头像标记 → 强制重置为默认头像。敏感昵称检测 → 强制修改。支持自动检测 + 人工复审双通道。
- **审核工作流通用能力**：待审核队列按优先级/时间排序。审核记录（谁/何时/什么操作/理由）。驳回时必填理由，通知提交者。批量审核（全选当前页通过/驳回）。

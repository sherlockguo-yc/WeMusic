# 数据看板

- **实时概览卡片**：DAU / MAU / 新增注册 / 活跃播放用户数。实时播放数、当前在线用户数。歌曲/歌单/反馈/屏蔽源总量。播放分钟数 24h/7d/30d 汇总。
- **趋势图表**：用户增长（折线图，日/周/月粒度）。播放时长时段分布热力图。注册→活跃转化漏斗。新用户 Day-1 / Day-7 / Day-30 留存率。反馈提交趋势（按 Bug/需求/其他分组）。
- **内容分析**：热门歌曲 Top 50（按播放次数 / 播放时长排序）。热门歌手 Top 20。高频搜索词排行。播放源健康度：B 站视频可用率、平均加载耗时。红心/收藏热门排行。
- **用户画像**：活跃度分层（高活 / 中活 / 低活 / 沉睡用户）。听歌偏好分布（流派 / 语种 / 时段偏好）。人均使用时长、人均播放歌曲数。
- **数据导出**：看板数据导出 CSV/JSON。定时报告：周报/月报自动生成并推送。自定义时间范围筛选。

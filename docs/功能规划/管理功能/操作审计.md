# 操作审计

- **管理员操作日志**：记录关键操作及其上下文 — 操作人、目标对象、操作类型、操作时间、IP地址、变更前后对比。支持的操作类型：用户管理、内容审核、系统配置变更、权限变更、备份恢复。
- **安全事件日志**：管理员登录/登出记录。敏感操作记录。异常检测（异地 IP 登录、非常规时段操作、短时间内大量 403 请求）。API 访问异常统计。
- **日志查询与导出**：按操作人/操作类型/时间范围/目标对象多条件筛选。导出为 CSV/JSON。日志不可被管理员删除或篡改（追加写入 + 哈希链）。
- **日志保留策略**：可配置保留天数，超期自动清理。在清理前可触发归档导出。

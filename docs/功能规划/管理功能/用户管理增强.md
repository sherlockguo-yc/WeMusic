# 用户管理增强

- **用户详情页**：完整播放历史（分页+搜索）、歌单及歌曲明细、反馈历史、登录历史（时间/IP/UA）、操作时间线（注册 → 首播 → 创建歌单 → 最近活跃）。
- **用户操作**：禁用/启用账号（禁用后立即踢出所有会话）。重置密码（生成临时密码或发送重置链接到绑定邮箱）。强制下线（撤销所有 JWT token）。编辑用户信息（修正用户名、清除违规头像）。
- **删除用户**：级联删除或匿名化关联数据。可选"先导出数据再删除"（合规需求）。须超级管理员二次密码验证。不可逆操作，有醒目警告。
- **批量操作**：条件筛选（注册时间范围 / 活跃状态 / 播放时长阈值 / 自定义标记），批量导出、批量禁用/启用、批量发送站内通知。
- **用户标记与备注**：管理员可为用户添加内部备注（仅管理员可见）。状态标记：正常 / 观察中 / 已警告 / 已封禁。按标记筛选用户列表。
- **列表增强**：服务端分页（避免大量用户数据撑爆前端）。可排序列（注册时间 / 播放时长 / 最近活跃）。搜索（用户名模糊匹配）。

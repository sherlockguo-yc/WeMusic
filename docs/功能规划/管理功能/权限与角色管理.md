# 权限与角色管理

- **角色体系**：预置角色 — 超级管理员（Super Admin，最高权限）/ 管理员（Admin，日常运营）/ 内容审核员（Moderator，仅审核）/ 只读观察员（Viewer，仅查看）。支持自定义角色，选择可见模块与操作权限。
- **管理员管理**：从现有用户中委任/移除管理员角色。管理员列表展示角色、最后活跃时间、操作概览。支持管理员自助放弃权限。管理员会话管理：查看与撤销其他管理员的活跃会话。
- **操作级权限控制**：每个模块内区分「查看」「编辑」「删除」「导出」等操作，可针对角色精细配置。数据级权限：特定管理员只能查看自己负责范围的数据。
- **安全机制**：管理员登录可配置二次验证（TOTP 验证码）。敏感操作（删除用户、重置密码、批量操作、备份恢复）须二次确认。异常登录告警（异地 IP、非常规时段）。

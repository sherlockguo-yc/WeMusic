# 系统配置

- **注册控制**：开放注册 / 仅邀请制 / 关闭注册（运行时切换，无需重启）。邀请码管理：批量生成、使用次数限制、有效期设置、使用记录查询。注册审批队列（管理员手动通过/拒绝，附理由）。IP 白名单/黑名单，邮箱域名限制。
- **功能开关（Feature Flags）**：运行时启用/禁用功能模块（歌词搜索、视频源切换、社交功能等）。灰度发布：按用户百分比或特定用户组逐步开启新功能。开关变更历史可追溯。
- **全局参数**：播放默认值（初始音量、淡入淡出时长）。API 请求频率限制（全局 + 单用户粒度）。缓存策略配置。推荐算法参数在线调整（多样性权重、新鲜度权重、冷门歌曲曝光率）。
- **公告管理**：创建/编辑/删除系统公告。公告展示位置：顶栏横幅 / 弹窗 / 播放页底部。定时发布 + 过期自动下架。目标用户范围：全部 / 新用户 / 活跃用户 / 自定义筛选。
- **备份与恢复**：手动触发完整数据库备份。每日自动备份（保留最近 N 份）。备份历史列表、下载备份文件。从备份恢复（须超级管理员验证，恢复前自动创建当前数据快照）。磁盘空间不足时拒绝备份并告警。

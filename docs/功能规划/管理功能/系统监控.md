# 系统监控

- **服务健康**：CPU / 内存 / 磁盘使用率实时展示。Node.js 进程指标（事件循环延迟、堆内存、GC 频率）。数据库连接池状态、慢查询日志。
- **API 性能**：各端点 P50 / P95 / P99 响应时间。QPS 实时趋势图。错误率监控（4xx / 5xx 分布）。第三方 API 调用成功率（QQ 音乐、B 站、网易云）。
- **后台任务**：任务状态面板（歌词抓取、视频源搜索队列长度）。失败任务告警与重试管理。单次任务耗时统计。
- **告警规则**：可配置阈值 — API 错误率 > 阈值 / 磁盘使用率 > 阈值 / 连续 N 分钟无播放请求 / 数据库连接池耗尽。告警通知方式（站内 / 邮件 / Webhook / 企业微信机器人）。

# 基于微信的社交增强 ⛔ 依赖「微信登录与账号绑定」

## 概述

在站内独立社交功能的基础上，接入微信体系带来的增强能力：微信好友关系导入、微信分享歌单/歌曲、微信扫码一键登录降低注册门槛。这些功能均依赖微信开放平台，当前因缺少开发者账号、备案域名等前置条件而暂时搁置。

## 功能方向

- **微信好友关系导入**：授权后导入微信好友列表，快速关注已在 WeMusic 的好友，向未注册好友发送邀请。
- **微信分享歌单/歌曲**：分享歌单或歌曲到微信聊天/朋友圈，好友点击直接跳转 WeMusic 播放。
- **微信快捷登录**：微信扫码一键登录/注册，降低新用户注册摩擦（详见「微信登录与账号绑定」）。

## 思考

这些功能是社交功能的「放大器」而非「发动机」——站内社交体系（评论/关注/动态/一起听）本身不依赖微信，微信仅用于降低获客和分享的门槛。建议先上线站内独立社交功能，待微信前提条件满足后再接入增强。
